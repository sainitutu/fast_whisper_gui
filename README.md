# fast_whisper_gui

## 使用前需安裝
- python(建議使用3.10.6)

## 用法

下載後，點擊 windows_start.bat執行(如果是linux，請啟用Linux_start.sh)

自動安裝虛擬環境與安裝參數

使用時選需要的參數，點下start後就能運行了

ps.目前不加速執行會出問題，待有時間再來修正
